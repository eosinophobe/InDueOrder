# Custom Card Queue

This RemNote plugin loads every card generated by a chosen Rem, shows the card IDs, and feeds those cards into a controlled custom queue.

## Features

- Load cards by pasting a Rem ID.
- Load cards from the currently focused Rem.
- Inspect each loaded card's card ID, source Rem ID, and card type.
- Review only those cards inside an embedded RemNote queue.

## Development

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev
```

Then, in RemNote, open Plugins -> Build -> Develop from localhost and enter:

```text
http://localhost:8080
```

## Build

```bash
npm run build
```

The production zip will be generated as PluginZip.zip.
